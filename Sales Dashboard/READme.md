# Sales Analysis Power BI Dashboard

## Project Overview

This project is a fictional sales analysis dashboard built in Power BI. It visualizes sales performance across various dimensions, including country, salesperson, product type, and time. The dashboard helps in identifying key sales trends, top-performing products, and the impact of different regions on total sales.

### 1. Sales by Country
   - **Top Countries by Sales**: UK (26.15%), USA (23%), India (16.67%), Canada, New Zealand, Australia.

## How to Use the Dashboard

1. **Filter by Country**: Use the country filter to view sales data specific to each country.
2. **Filter by Salesperson**: Select a salesperson from the filter to examine their individual performance.
3. **Filter by Product**: Explore sales performance for specific chocolate products.
4. **Date Range Selector**: Adjust the date range to analyze sales trends for a specific period.

## Tools & Technologies Used

- **Power BI Desktop**: Used for data modeling, visualization, and dashboard creation.
- **Data Transformation**: Power Query was used for data cleaning and transformations.
- **DAX Measures**: Custom DAX measures for calculating total sales, top products, and other key metrics.
